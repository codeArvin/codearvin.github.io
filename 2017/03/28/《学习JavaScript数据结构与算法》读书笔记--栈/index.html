概念

栈是一种遵循后进先出(LIFO)的有序集合。新添加的或待删除的元素保存在栈的末尾，称为栈顶，另一端称作栈底。在栈中，新元素都靠近栈顶，旧元素靠近栈底

汉诺塔问题

1、概念: 从左到右有A、B、C三根柱子，其中A柱子上面有从小叠到大的n个圆盘，现要求将A柱子上的圆盘移到C柱子上去，期间只有一个原则：一次只能移到一个盘子且大盘子不能在小盘子上面，求移动的步骤和移动的次数

2、原理: 运用递归，可以将步骤抽象为：

  • (1): 先将A柱的n-1个移动到B柱
  • (2): 再将A柱的最后一个移动到C柱
  • (3): 最后再将B柱的n-1个移动到C柱
